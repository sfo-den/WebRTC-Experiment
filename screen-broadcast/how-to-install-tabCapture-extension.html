How to install tabCapture extension?

This document explains how to broadcast your screen over many peers using Google Chrome tabCapture extension APIs.
« Go Back to Screen Broadcasting! or Download extension (ZIP file)
First of all; you need to install Google Chrome Canary and then Download ZIP.
  1. Check "Developer mode" option and click "Load unpacked extension..." button and then locate extension folder (directory).
  2. Open tab you want to broadcast and click icon to start broadcasting your screen.
  3. Otherwise, right-click; open context-menu and select "Broadcast Screen" option.
  4. After broadcasting, you can see your room is listed in this page.
If you want to broadcast screen privately; open "broadcaster.js" file; goto Line 2 and uncomment the commented lines. Then press "Ctrl + R" or click "Reload" button/link in the extension page (chrome://extensions).
