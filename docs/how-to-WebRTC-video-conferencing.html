How to video conferencing?

  1. Open multiple peers to support multi-user connectivity
  2. Open multiple sockets to exchange SDP/ICE among mulitple users
and that's all!

You JUST need to call "conference" object and pass a few methods/objects and that's all you need to do!
var conferenceUI = conference(config);
And here is how you should pass options/config: 
var config = {
    openSocket : function (socket_config) {
        var socket = io.connect('...');
        
        socket.on('connect', socket_config.onopen);
        socket.on('message', socket_config.onmessage);

        return socket;
    },
    onRemoteStream : function (media) {
        document.body.appendChild(media.video);
    },
    onRoomFound : function (room) {
        // room.roomName
        // room.broadcaster
        // room.roomToken
    },
    attachStream: 'client stream'
};
To create a new conference room: 
conferenceUI.createRoom({
    roomName : 'Anonymous'
});
To join a room: 
conferenceUI.joinRoom({
    roomToken : 'unique token of the room',
    joinUser : 'unique token of the room owner/broadcaster'
});
conference-ui.js is a good place to start!

How openSocket works?
In this function, you JUST need to write socket.io relevant code that will be called by conference.js many times.
What you need to do is JUST open socket and get messages passed over that socket.
Then you should call onopen and onmessage accordingly.

How onRemoteStream works?
The function will be called each time when remote stream starts flowing.
It will be called for each conference participant.
conference.js will return two objects: 1) media.video and 2) media.stream

How onRoomFound works?
When a user opens your webpage; you can show him list of available public/private conference rooms.
Following 3 objects will be passed over this method:
  1. roomName: The human readable conference room name.
  2. broadcaster: The person who creates the conference room. You can say him the "owner" of the room.
  3. roomToken: Unique ID of the room.
You'll use broadcaster and roomToken when joining a room. So these two values are mandatory.

What is attachStream?
Obviously, your client media stream (webcam + microphone) to be attached for remote peer(s).
