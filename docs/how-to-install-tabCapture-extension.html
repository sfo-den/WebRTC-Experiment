How to install tabCapture extension?

Copyright © 2013 Muaz Khan<@muazkh>.

First of all; you need to install Google Chrome Canary or Dev Channel and then Download ZIP.
Goto "chrome://extensions/", check "Developer mode" option and click "Load unpacked extension..." button and then locate extension folder (directory).
Open tab you want to share and click top-right icon to start broadcasting your tab.
Otherwise, right-click; open context-menu and select "Share this tab!" option.
After broadcasting, you can see your tab is listed here.
