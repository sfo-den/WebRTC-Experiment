Rules / Privacy
WebRTC Experiment!

Privacy

1. Chat is 100% secure in this app! Currently this app is not using WebRTC DataChannel APIs to share chat message among peers. However, you'll see it in action as soon as Google Chrome support DataChannel APIs.
2. Each chat message is quickly and automatically deleted from the server (in a few seconds)!
3. Private rooms are only visible to your real friends!

Rules

1. You are not allowed to use this app for illegal activities.
2. You MUST have installed this app's compatible web browsers like Google Chrome 23.x and upper, Mozilla Firefox etc.
3. You are not allowed to copy this app's code and declare as your own property. A back-link is required in this situation.
