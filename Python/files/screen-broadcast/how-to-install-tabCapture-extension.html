How to install tabCapture extension?

This document explains how to broadcast your screen over many peers using Google Chrome experimental tabCapture extension APIs.
« Go Back to Screen Broadcasting! or Download extension (ZIP file)
First of all; you need to install Google Chrome Canary.
  1. Check "Developer mode" option and click "Load unpacked extension..." button and then locate extension folder (directory).
  2. Open tab you want to broadcast and click "red" icon to start broadcasting your screen.
