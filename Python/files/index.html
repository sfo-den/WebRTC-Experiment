WebRTC Screen Broadcasting!

WebRTC screen broadcasting: Using Chrome Experimental tabCapture APIs to broadcast screen over many peers.
You need to download extension (in ZIP format) if you want to broadcast your screen. To join any broadcasted screen; you don't need to download ZIP file.
Note: this experimental chrome extension only works with latest Google Chrome Canary.
