RTCMultiConnection API Reference / "renegotiate" method

  1. "renegotiate" method is useful to retry/renegotiate with or without adding additioanl method streams! 
  2. connection.renegotiate(optional_Custom_MediaStream);
  3. This method renegotiates among all users in the section. To renegotiate between two unique users; try "peers" object: 
  4. connection.peers['user-id'].renegotiate(optional_Custom_MediaStream);
