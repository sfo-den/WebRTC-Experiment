  1. RTCMultiConnection.js: A comprehensive wrapper library for RTCWeb APIs; fits in any environment!
  2. RTCMultiConnection Videos Channel!
  3. You can use RTCMultiConnection.js in any webpage; on any environment; in any framework; in any situation; in any style! It is highly customizable; very rich; and totally free WebRTC Library!
  4. You can use it in any PHP project; in any ASP.NET project; in any Ruby project; anywhere!
  5. RTCMultiConnection.js doesn't care about signaling gateways; it is upto you to decide whether you want to use MySQL, MS-SQL or WebSockets, WebSync, SignalR, or SignalMaster or PeerServer or SIP, XMPP, or any thing else exists in the globe! Read more here!

  6. Want to use RTCMultiConnection.js?

    1. First step you should do is link the library file in your HTML/PHP page.
    2. Second step you should do is override openSignalingChannel method. You can override it in your HTML page. This method allows you choose 3rd party services like Firebase, PubNub, Pusher etc. or open-source services like PeerServer, SignalMaster or custom services like WebSockets, Socket.io, SignalR, WebSync, or famous voip services like SIP and XMPP or traditional services like XHR/JSON/MySQL.

  9. Setup Video Conference using full syntax!

    <script src="http://www.rtcmulticonnection.org/latest.js"></script>
<button id="openNewSessionButton">Open New Room</button>
<script>

    var connection = new RTCMultiConnection();

// easiest way to customize what you need!
connection.session = {
    audio: true,
    video: true
};

// on getting local or remote media stream
connection.onstream = function(e) {
    document.body.appendChild(e.mediaElement);
};

// setup signaling channel
connection.connect();

// open new session
document.querySelector('#openNewSessionButton').onclick = function() {
    connection.open();
};

    </script>

  10. Setup Multi-user file sharing along with advance progress-bars!

    <script src="http://www.rtcmulticonnection.org/latest.js"></script>
<input type=file>
<button id="openNewSessionButton">Open New Room</button>
<script>

    var connection = new RTCMultiConnection();

connection.session = {
    data: true  // this line suggests that don't get/use audio/video; 
                // only setup data connection
};

// setup signaling channel
connection.connect();

// open new session
document.querySelector('#openNewSessionButton').onclick = function() {
    connection.open();
};

// send single file or multiple files concurrently!
document.querySelector('input[type=file]').onchange = function() {
    connection.send(this.files[0]);
};

    </script>
