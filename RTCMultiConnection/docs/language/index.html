RTCMultiConnection API Reference / "language" object

  1. Using "language" object; you can set target language in which text or voice should be auto converted. 
  2. connection.language = 'ja'; // prefer Japanese
connection.autoTranslateText = true;

// you can use "e.data" and "e.original"
connection.onmessage = function(e) {
   // e.data     ---- translated text
   // e.original ---- original text
};
