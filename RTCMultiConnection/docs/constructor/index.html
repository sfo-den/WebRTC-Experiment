RTCMultiConnection API Reference / Constructor

  1. RTCMultiConnection constructor takes single argument; named as "channel-id". 
    var connection = new RTCMultiConnection('channel-id');
  2. If you skip "channel-id"; RTCMultiConnection will use "full-URL" as channel-id: 
    var channel_id = location.href.replace(/\/|:|#|%|\.|\[|\]/g, '');
var connection = new RTCMultiConnection(channel_id);
  3. You can access "channel-id" like this: 
    console.log('channel-id is', connection.channel);
  4. You can set "channel-id" like this as well: 
    connection.channel = 'channel-id';

Reference

  1. http://www.rtcmulticonnection.org/docs/channel-id/
