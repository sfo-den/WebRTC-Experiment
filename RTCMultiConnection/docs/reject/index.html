RTCMultiConnection API Reference / "reject" method

  1. In "admin/guest" scenarios; a person can reject request of a user using "reject" method. 
  2. connection.onRequest = function (request) {
    // either accept
    connection.accept(request);
    
    // or reject
    connection.reject(request);
};
  3. You can use onstats to known whether request is either rejected or accept 
  4. connection.onstats = function (stats, callee) {
    // callee rejected the request
    if (stats == 'rejected') {}

    // callee accepted caller's request
    if (stats == 'accepted') {}
    
    // callee.userid || callee.extra
};
